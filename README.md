# 🎓 ExamApp Management System
A full-featured Online Examination System built with **Django** and **MySQL**, featuring a modern Bootstrap 5 UI.

## 🚀 Features
- **Student Dashboard:** View stats, take exams, and see real-time results.
- **Admin Panel:** Add/Delete questions and manage student records.
- **Dynamic Scoring:** Real-time calculation with pass/fail visual feedback.
- **Responsive UI:** Mobile-first design for taking tests on any device.

## 🛠️ Tech Stack
- **Backend:** Django (Python)
- **Database:** MySQL
- **Frontend:** Bootstrap 5, HTML5, CSS3
